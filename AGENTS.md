# ArgosMate

Simple macOS menu bar app that connects to an Argos espresso machine via Bluetooth (BLE) and displays real-time temps in the menu bar.

## Tech Stack

- SwiftUI (menu bar app)
- CoreBluetooth (BLE connection)
- No external dependencies (no SPM, CocoaPods, or Carthage)

## Notes

- Requires Bluetooth permission (`NSBluetoothAlwaysUsageDescription` in Info.plist)
- No test files exist in this repo
